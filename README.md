# PGBooks

### PGBooks

- objective-c 버전
- 기본 mvc 패턴 구조
- cocoapods


### PGBooks-swift

- swift 버전

- cocoapods

### PGTransitionKit

- swift based.
- objective-c 프로젝트 지원.
- 기존 프로젝트에서 objc 지원되도록 버그픽스 진행됨.

#### etc
- xcode 11 , xcode10.3 에서 테스트 진행.
